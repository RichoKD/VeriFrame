# FluxFrame Color Palette

## Primary Colors

### Blues
- `#2E90E0` - `blue-600` (Primary brand blue)
- `#0EA5E9` - `blue-500` (Bright blue accent)
- `#1E40AF` - `blue-700` (Dark blue hover state)

### Cyans
- `#06B6D4` - `cyan-500` (Cyan accent)
- `#22D3EE` - `cyan-400` (Light cyan)
- `#164E63` - `cyan-900` (Dark cyan)

### Purples
- `#A855F7` - `purple-500` (Purple accent)
- `#6B46C1` - `purple-700` (Deep purple)
- `#9333EA` - `purple-600` (Medium purple)

## Neutral Colors (Dark Theme)

### Zinc/Gray Scale
- `#09090B` - `zinc-950` (Almost black background)
- `#18181B` - `zinc-900` (Dark background)
- `#27272A` - `zinc-800` (Card/section background)
- `#3F3F46` - `zinc-700` (Borders)
- `#52525B` - `zinc-600` (Muted text)

### Slate
- `#0F172A` - `slate-900` (Very dark)
- `#1E293B` - `slate-800` (Dark background)
- `#334155` - `slate-700` (Muted text)
- `#64748B` - `slate-500` (Medium gray text)
- `#94A3B8` - `slate-400` (Light gray text)
- `#CBD5E1` - `slate-300` (Light text)
- `#E2E8F0` - `slate-200` (Very light text / headings)
